Menu bar utility 菜单栏工具

Switch ChatGPT accounts in one click. 一键切换 ChatGPT 账号。

CX Switch lives in your menu bar: instant switching for ChatGPT / Codex, live usage bars, and per-account credentials in a local SQLite store. CX Switch 常驻菜单栏：为 ChatGPT / Codex 提供极速切换、实时用量条，以及按账户保存的本地 SQLite 凭证。

Features 功能亮点

Built for fast switching and clear usage. 为快速切换与清晰用量而设计。

How it works 工作流程

Three steps. No juggling browser tabs. 三步完成，不再来回切换浏览器。

  1. 1

    Add accounts 添加账户

    Sign in to each ChatGPT account with the built-in auth flow. 通过内置登录流程，逐个添加 ChatGPT 账户。

  2. 2

    Click switch 点击切换

    Pick an account from the menu bar — switching is instant. 在菜单栏选择目标账户——切换几乎是瞬时的。

  3. 3

    Codex uses the new account Codex 使用新账户

    Active credentials are written to ~/.codex/auth.json so Codex picks them up automatically. 当前凭证会写入 ~/.codex/auth.json，Codex 会自动读取。

FAQ 常见问题

Answers to common questions. 常见问题解答。

Is CX Switch free? CX Switch 免费吗？ ＋

Yes. CX Switch is open source under Apache-2.0 and has no in-app purchases.

是的。CX Switch 在 Apache-2.0 下开源，没有应用内购买。

Where are credentials stored? 凭证存在哪里？ ＋

Locally in an encrypted SQLite database under Application Support: ~/Library/Application Support/com.novainfra.cx-switch/cx-switch.db.

凭证加密保存在本机 Application Support 下的 SQLite： ~/Library/Application Support/com.novainfra.cx-switch/cx-switch.db。

Does it work with ChatGPT Teams or Enterprise? 支持 ChatGPT Team / Enterprise 吗？ ＋

The app targets standard ChatGPT accounts. Team / Enterprise compatibility depends on your org’s auth setup — try it and file an issue if something breaks.

应用面向常规 ChatGPT 账户。Team / Enterprise 取决于组织的鉴权配置——如遇问题欢迎提 issue。

Is any data sent to third-party servers? 会向第三方服务器发送数据吗？ ＋

CX Switch does not ship your credentials to a third-party analytics service. Tokens are exchanged directly with OpenAI’s auth endpoints when you sign in.

CX Switch 不会把凭证发送到第三方统计服务。登录时 token 直接与 OpenAI 鉴权端点交换。

How does Codex pick up the active account? Codex 如何识别当前账户？ ＋

When you switch accounts, CX Switch writes the active credentials to ~/.codex/auth.json so the Codex CLI uses the new session.

切换账户后，CX Switch 会把当前凭证写入 ~/.codex/auth.json，Codex CLI 将使用新的会话。

Is the source code available? 源码是否公开？ ＋

Yes — the repository is public on GitHub so you can audit behavior and build from source.

是的——仓库在 GitHub 公开，你可以审计行为或自行编译。

What macOS versions are supported? 支持哪些 macOS 版本？ ＋

macOS 14 (Sonoma) or later. Apple Silicon and Intel are supported.

macOS 14（Sonoma）或更新；支持 Apple Silicon 与 Intel。
