Product

A compact tool for OpenAI account switching.

The current app is a macOS menu bar utility, and these are the features that are visible in the codebase today.

Menu bar only

CX Switch launches as a macOS menu bar app with no Dock icon.

Current and saved accounts

The panel shows the current account and a list of saved accounts for quick switching.

Usage and reset timers

The current account card includes usage bars and reset countdowns when the data is available.

Import and language support

Users can import a refresh token, open settings or status, and switch the UI between Chinese and English.

Install

Run the app locally.

The project is an Xcode app built from a Swift package, so setup is straightforward on macOS 14+.

1

Open the project

Use Xcode 15+ on macOS 14 or later, then open CXSwitch.xcodeproj.

2

Run the app

Launch CXSwitch and let it appear in the menu bar. The app loads your dashboard and account data automatically.

3

Use the panel

Add accounts, switch between them, refresh usage, or import a refresh token from the menu bar panel.
