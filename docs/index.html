What GPT-5.4 teaches us

Design starts with a story, not with chrome.

The OpenAI article argues that stronger frontends come from better constraints, clearer content, and visual verification. We turned that advice into the structure of this page.

01

Define constraints up front

Choose the layout, the palette, the hierarchy, and the tone before adding details. Good output is easier when the model sees the rules first.

02

Use a narrative page flow

Hero, support, detail, and final action. That rhythm keeps the page focused and prevents generic dashboard sprawl.

03

Ground the page in real content

Real product copy, real usage states, and real install steps make the site feel believable instead of decorative.

04

Verify on desktop and mobile

The article emphasizes inspection and iteration. This page uses a responsive hero, readable spacing, and motion that respects reduced motion settings.

Current features

Everything the current app already does.

These are drawn from the SwiftUI rewrite in this repository, so the site reflects the app as it exists now.

Menu bar first

CX Switch runs as a macOS menu bar app with no Dock icon, keeping account switching close to where you need it.

Usage at a glance

The app shows your current account, plan badge, 5-hour usage, weekly usage, and reset countdowns in one compact panel.

Fast account actions

Add, save, switch, refresh, and import refresh tokens without leaving the menu bar.

Bilingual ready

UI strings are centralized and can be shown in Chinese or English, which makes the product easier to localize and maintain.

Install

Run it locally in a few steps.

The repository is a Swift package and Xcode project, so local setup stays simple.

1

Open the project

Use Xcode 15+ on macOS 14 or later, then open CXSwitch.xcodeproj.

2

Run the app

Launch CXSwitch and let it appear in the menu bar. The app loads your dashboard and account data automatically.

3

Start managing accounts

Add an account, import a refresh token, or switch to a saved account from the menu bar panel.
